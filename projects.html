Discord Embed Forwarder

Python • discord.py • Webhooks • Requests

Listens to a source channel for webhook-based messages and forwards the sanitized embed(s) to multiple destination webhooks. Useful for #live-checkouts mirroring between servers. Secrets are handled via environment variables.

Key Features

  • Forwards only embeds (skips regular chat noise)
  • Whitelists important fields (Original Price, Retail Price, Quantity)
  • Optional announcement field on first forwarded message
  • Environment-based config via .env
pip install discord.py requests python-dotenv

# .env
DISCORD_BOT_TOKEN=your_bot_token_here
TARGET_CHANNEL_ID=123456789012345678
DESTINATION_WEBHOOKS=https://discord.com/api/webhooks/xxx,https://discord.com/api/webhooks/yyy

# run
python assets/code/discord_forwarder.py

Inventory Desktop App

Electron • JavaScript • SQLite

Desktop tool for resellers to track inventory, costs, and sales with fast CRUD, real-time analytics, and a clean dark UI.

Highlights

  • SQLite schema for items, banks, and sales with stock counting
  • Editable forms with dropdown for banks from Settings
  • Dark mode, sidebar navigation, and barcode scanning hook

Sample SQL Schema

-- Inventory table with cost, predicted vs actual resell, and sold tracking
CREATE TABLE IF NOT EXISTS inventory (
    id               INTEGER PRIMARY KEY AUTOINCREMENT,
    timestamp        TEXT    DEFAULT CURRENT_TIMESTAMP,
    sku              TEXT    NOT NULL,
    shoe_name        TEXT    NOT NULL,
    size             TEXT    NOT NULL,
    card             TEXT,
    retail           REAL,
    predicted_resell REAL,
    resell           REAL,
    quantity         INTEGER DEFAULT 1,
    date_sold        TEXT
);

-- Analytics query: total sales and profit, filterable by date range
SELECT
    COALESCE(SUM(resell), 0)          AS total_sales,
    COALESCE(SUM(resell - retail), 0) AS total_profit
FROM inventory
WHERE date_sold IS NOT NULL
  AND date_sold >= date('now', '-12 months');

Stock & Crypto Alert Bot

Python • Yahoo Finance API • CoinGecko API • Discord Webhooks

Monitors stock and cryptocurrency prices using public APIs and sends real-time Discord alerts when user-defined thresholds are crossed. Price data is stored persistently so the bot can track trends across sessions.

Key Features

  • Pulls live stock prices via Yahoo Finance API and crypto prices via CoinGecko API
  • Sends formatted Discord alerts when price goes above or below a set threshold
  • Persists watched assets and alert history to a local database
  • Configurable via .env — webhook URL, tickers, and thresholds
  • Scheduled polling loop runs continuously on Raspberry Pi
# Example config (.env)
DISCORD_WEBHOOK=https://discord.com/api/webhooks/xxx
WATCH_STOCKS=AAPL,NVDA,TSLA
WATCH_CRYPTO=bitcoin,ethereum
ALERT_THRESHOLD_PCT=5

# run
python stock_alert_bot.py

Target Automation Tool

Python • Playwright • Proxy Rotation • OTP Handling

A browser automation tool built with Playwright that handles full login flows including OTP (one-time password) authentication, proxy rotation for session management, and offer/deal scraping from Target's platform.

Key Features

  • Automates login flow including email/password and OTP two-factor authentication
  • Proxy rotation support to manage multiple sessions independently
  • Scrapes available offers and logs them for review
  • Headless mode for running on a server or Raspberry Pi
  • Environment-based config via .env for credentials and proxy list
pip install playwright python-dotenv
playwright install chromium

# .env
TARGET_EMAIL=your_email
TARGET_PASSWORD=your_password
PROXY_LIST=ip1:port,ip2:port

# run
python target_automation.py
